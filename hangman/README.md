# Hangman

A simple hangman game.

## Getting Started

Clone the project.

```
git clone https://github.com/bankkeez/internship.git
```

## Prerequisites

* Java SE 8 or above.

## Running the program

After fininsh cloning, go to the jar file destination.

```
cd internship/hangman
```

This command runs the jar file.

```
java -jar hangman.jar
```
